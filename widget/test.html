Testing RAG Chat Widget

This page demonstrates how the RAG Chat Widget can be embedded into any website with a single script tag.

Try clicking on the chat bubble in the bottom-right corner to start a conversation.

How It Works

The widget connects to a backend FastAPI server that implements a Retrieval Augmented Generation (RAG) system using Azure OpenAI and AstraDB.

You can ask questions about documents that have been ingested into the system, and the AI will respond based on the content of those documents.
