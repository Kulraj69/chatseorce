RAG Chat Widget Embedding Example

This page demonstrates how to embed the RAG Chat Widget in any website.

How to Embed

Copy and paste the following code into your HTML page:

<!-- RAG Chat Widget -->
<div id="chat-widget-root"></div>
<script src="https://yourdomain.com/path/to/chat-widget.js" defer></script>

That's it! The chat bubble will appear in the bottom-right corner of your website.

Widget Configuration

To configure the API endpoint, edit the src/api.js file and change the API_BASE_URL before building:

// src/api.js
const API_BASE_URL = 'http://localhost:8080';
