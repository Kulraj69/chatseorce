RAG Chat Widget: Production Embedding

Basic Embedding

Copy this minimal code to embed the widget:

<script src="https://your-cdn.com/chat-widget.js" defer></script>

Advanced Embedding with Configuration

For more control, you can configure the widget before loading it:

<!-- Widget Configuration -->
<script>
  window.RAG_WIDGET_CONFIG = {
    api: {
      baseUrl: 'https://api.yourdomain.com', // Your API endpoint
      timeout: 30000 // milliseconds
    },
    widget: {
      title: 'Company Assistant',
      position: 'bottom-right',
      initiallyOpen: false
    },
    theme: {
      primaryColor: '#007bff', // Company brand color
      assistantBubbleColor: '#007bff'
    }
  };
</script>

<!-- Widget Script -->
<script src="https://your-cdn.com/chat-widget.js" defer></script>

Event Handling

You can listen to widget events:

<script>
  window.addEventListener('ragWidgetLoaded', function() {
    console.log('RAG Widget loaded successfully');
  });
  
  window.addEventListener('ragWidgetMessageSent', function(event) {
    console.log('User sent message:', event.detail.message);
    // Track in analytics, etc.
  });
  
  window.addEventListener('ragWidgetOpen', function() {
    console.log('Widget opened');
  });
</script>

Custom Triggers

You can open the widget programmatically:

<!-- Button to trigger widget -->
<button onclick="window.openRagWidget()">Chat with AI</button>

<script>
  // Define function before widget loads
  window.openRagWidget = function() {
    // This will be replaced by actual function when widget loads
    console.log('Widget not loaded yet');
  };
</script>
